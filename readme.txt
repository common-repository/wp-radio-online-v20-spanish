Plugin para incluir emisoras de radio en tu blog.

== Description ==
Emisoras del plugin: Años 80-90, Punk, Clasica, Dance, Alternativa, Hip Hop, House, Latino, Variado, Oriental, Popular, Relax, Rock.

== Installation ==

    1. Descargar el plugin, descomprimir y subir al directorio: /wp-content/plugins/.
    2. Entra en tu panel de administración de Wordpress, sección plugins y activa el wp-radio-plugin.
    3. Ahora ya puedes configurarlo en la sección de opciones y posteriormente, añadir el widget.
